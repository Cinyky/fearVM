# fearVM
A javascript virtual machine protector

## Concept
fearVM compiles javascript to bytecodes, then run it with a virtual machine

## Supported JS Syntax
* Almost everything in es5, except try-catch-break
* Some of ES6
  * arrow functions
  * let & const
  * Spread syntax (for function and array)
